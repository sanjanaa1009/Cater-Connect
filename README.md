# 🍽️ Catering Marketplace

A full-stack web app to discover and book local caterers.

## 🚀 Features
- Browse caterers
- View menus
- Add to cart
- Book catering services
- Location-based discovery (coming soon)

## 🛠️ Tech Stack
- Frontend: React + Tailwind
- Backend: Node.js + Express
- Database: MongoDB
- Auth: JWT (planned)
- Realtime: Socket.io (planned)

## 📦 Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## 🔐 Environment Variables
Create `.env` in backend:

MONGO_URI=your_mongo_url
